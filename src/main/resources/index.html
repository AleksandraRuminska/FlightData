About the project

The Flights Info application allows to find:

  1. in the subpage flights given the flight number and the date:
    • total weight of baggage
    • total weight of cargo
    • total weight overall
  2. in the subpage airports given the airport code and date:
    • number of arrivals
    • number of departures
    • number of baggage that arrived
    • number of baggage that departed
